📖 AI & LLM Glossary

A collection of common AI, LLM, and Generative AI terms explained in simple language.

«Tip: Use "Ctrl + F" to quickly find a term.»

---

A

Agent

An AI system that can reason, plan, and use tools to complete tasks.

API

Application Programming Interface. A way for software applications to communicate with each other.

Attention

The mechanism that allows an LLM to focus on the most relevant parts of the input when generating text.

---

C

Chunking

The process of splitting large documents into smaller pieces before storing them for retrieval.

Context

The information provided to an LLM before it generates a response.

Context Engineering

Designing and managing the information sent to an LLM to improve response quality.

Context Window

The maximum amount of information (tokens) an LLM can process in a single request.

---

E

Embedding

A numerical representation of text that captures its meaning, making similarity search possible.

---

F

Fine-tuning

Training a pre-trained model on additional data to specialize it for a specific task.

Function Calling

A feature that allows an LLM to invoke external functions or APIs.

---

G

Generative AI

AI systems that create new content such as text, images, audio, video, or code.

Guardrails

Rules and safety mechanisms that help keep AI outputs reliable and appropriate.

---

H

Hallucination

When an LLM generates information that sounds convincing but is incorrect or fabricated.

---

L

LLM (Large Language Model)

A machine learning model trained on massive amounts of text to understand and generate human language.

---

M

MCP (Model Context Protocol)

An open protocol that standardizes how AI models connect to external tools, data sources, and applications.

Multi-Agent System

A system where multiple AI agents collaborate to solve complex tasks.

---

P

Prompt

The instruction or input given to an AI model.

Prompt Chaining

Connecting multiple prompts together so the output of one becomes the input of another.

Prompt Engineering

The practice of writing effective prompts to achieve better AI responses.

---

R

RAG (Retrieval-Augmented Generation)

A technique that retrieves relevant information from external sources before an LLM generates its response.

Reasoning Model

An AI model designed to perform more structured reasoning on complex problems.

Retrieval

The process of finding relevant information from a knowledge base or database.

Reranking

Reordering retrieved results so the most relevant information appears first.

---

T

Token

The basic unit of text processed by an LLM. A token may be a word, part of a word, punctuation, or symbols.

Tool Calling

Allowing an LLM to use external tools, APIs, databases, or services while completing a task.

---

V

Vector

A list of numbers representing data mathematically.

Vector Database

A database optimized for storing and searching vector embeddings.

---

W

Workflow

A sequence of steps used to automate an AI-powered task.
